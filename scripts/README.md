# Scripts

This folder is reserved for project scripts and the Harness CLI.

Expected Harness CLI path after installation:

```bash
scripts/bin/harness-cli
```

The CLI was installed after the initial manual migration. Current verified command:

```bash
scripts/bin/harness-cli --version
```

Durable setup commands:

```bash
scripts/bin/harness-cli init
scripts/bin/harness-cli import brownfield
scripts/bin/harness-cli query matrix
```
