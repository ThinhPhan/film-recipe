# Harness Docs

This folder is the agent-facing operating surface for Film Recipe.

Start here:

- `docs/HARNESS.md`: collaboration model and source hierarchy.
- `docs/FEATURE_INTAKE.md`: how new work is classified.
- `docs/ARCHITECTURE.md`: current brownfield architecture map.
- `docs/TEST_MATRIX.md`: behavior-to-proof control panel.
- `docs/product/`: living product contract.
- `docs/stories/`: story packets and backlog.
- `docs/decisions/`: durable decisions.
- `docs/templates/`: reusable story, decision, and validation templates.

The legacy Specify output remains in `specs/` for audit/history. Ongoing work should update this Harness surface instead of extending the old monolithic specs.
